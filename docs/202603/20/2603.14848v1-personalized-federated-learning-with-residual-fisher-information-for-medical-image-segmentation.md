## 摘要
联邦学习使多个客户端（机构）能够在不共享私有数据的情况下协作训练机器学习模型。为了应对客户端之间数据异质性的挑战，个性化联邦学习 (pFL) 旨在为每个客户端学习定制化模型。在这项工作中，我们提出了 pFL-ResFIM，这是一种在参数级别实现客户端自适应个性化的新型 pFL 框架。具体而言，我们引入了一种新的度量指标——残差 Fisher 信息矩阵 (ResFIM)，用于量化模型参数对领域差异的敏感性。为了在隐私约束下估计每个客户端模型的 ResFIM，我们采用了一种频谱迁移策略，生成反映不同客户端领域风格的模拟数据。基于估计的 ResFIM，我们将模型参数划分为领域敏感和领域不变组件。然后，通过在服务器上仅聚合领域不变参数，为每个客户端构建个性化模型。在公开数据集上的广泛实验表明，pFL-ResFIM 始终优于现有最先进的方法，验证了其有效性。

## Abstract
Federated learning enables multiple clients (institutions) to collaboratively train machine learning models without sharing their private data. To address the challenge of data heterogeneity across clients, personalized federated learning (pFL) aims to learn customized models for each client. In this work, we propose pFL-ResFIM, a novel pFL framework that achieves client-adaptive personalization at the parameter level. Specifically, we introduce a new metric, Residual Fisher Information Matrix (ResFIM), to quantify the sensitivity of model parameters to domain discrepancies. To estimate ResFIM for each client model under privacy constraints, we employ a spectral transfer strategy that generates simulated data reflecting the domain styles of different clients. Based on the estimated ResFIM, we partition model parameters into domain-sensitive and domain-invariant components. A personalized model for each client is then constructed by aggregating only the domain-invariant parameters on the server. Extensive experiments on public datasets demonstrate that pFL-ResFIM consistently outperforms state-of-the-art methods, validating its effectiveness.

*   **研究动机**：
    *   现有的个性化联邦学习（pFL）方法（如 FedBN, FedPer）通常采用**层级（Layer-level）**的参数分离策略（例如固定保留 Batch Normalization 层或分类层在本地），这种启发式方法是静态且次优的。
    *   模型在训练过程中是动态的，不同参数对领域差异的敏感度不同，因此需要一种**原子级（参数级）**且**自适应**的个性化方案，以平衡知识共享与本地定制。

### 2. 核心方法论
该论文提出了 **pFL-ResFIM** 框架，其核心思想是利用 Fisher 信息矩阵来识别并保护领域敏感参数。
*   **残差 Fisher 信息矩阵 (ResFIM)**：
    *   引入 ResFIM ($\Delta F_W$) 来量化参数对领域差异的敏感性。它通过计算模型在本地数据分布与模拟的其他领域数据分布下 Fisher 信息矩阵（FIM）的差异来定义。
    *   高 ResFIM 值意味着该参数在跨领域时表现出高度的不稳定性，应视为“领域敏感”参数。
*   **隐私保护的频谱迁移策略**：
    *   为了在不泄露隐私的前提下估计跨领域 FIM，客户端在初始化时上传其图像的幅度谱（Amplitude Spectrum）到服务器并共享。
    *   客户端通过傅里叶变换（FFT）将本地图像的低频成分替换为其他领域的幅度谱，生成具有其他领域风格的模拟图像，从而在本地计算 ResFIM。
*   **参数级聚合机制**：
    *   根据 ResFIM 值，每个客户端生成一个二值掩码（Mask），选择前 $\delta\%$ 最敏感的参数保留在本地。
    *   服务器仅对掩码之外的“领域不变”参数进行聚合，从而实现精准的知识共享。

### 6. 主要结论与发现
*   **性能领先**：pFL-ResFIM 在所有数据集和几乎所有客户端上均优于 SOTA 方法，平均 Dice 分数提升显著。

### 7. 优点（亮点）
*   **粒度更细**：从传统的“层级”个性化提升到了“参数级”个性化，能够更精细地处理领域差异。
*   **动态自适应**：无需人工指定哪些层需要本地化，模型根据数据特性自动学习参数的重要性。
*   **隐私与效能兼顾**：通过频谱迁移模拟领域风格，既避免了原始数据交换，又解决了本地无法感知全局分布的问题。

*   **计算开销**：尽管使用了对角近似，但每个客户端仍需计算 FIM 并在本地生成模拟数据，这比 FedAvg 增加了额外的计算负担。

*   **通信开销**：除了模型参数，客户端还需要向服务器传输掩码矩阵（虽然是二值的），在极端带宽受限的情况下可能存在压力。
